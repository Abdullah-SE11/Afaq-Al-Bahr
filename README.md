# Horizons | Afaq Al Bahr Shipping L.L.C.

A premium 3D landing page for Afaq Al Bahr Shipping, featuring a stylized shipping vessel and ocean waves powered by Three.js.

## Tech Stack
- **Framework**: React + Vite
- **3D Engine**: Three.js & @react-three/fiber
- **Animations**: Framer Motion
- **Styling**: Vanilla CSS with Glassmorphism
- **Icons**: Lucide React

## Getting Started
1. `npm install`
2. `npm run dev`
